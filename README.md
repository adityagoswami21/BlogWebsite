# Flask Blog Website

A simple and elegant blog website built using **Flask**, **SQLAlchemy**, and **Bootstrap**. This project allows users to read blog posts, and admins can add, edit, and delete posts from a clean dashboard interface.

## Features

- View all blog posts on the homepage
- Admin dashboard for managing posts
- Create, edit, and delete blog posts
- Responsive UI using Bootstrap
- SQLAlchemy ORM for database interactions
- Environment variables support using `python-dotenv`

## Tech Stack

- Python
- Flask
- Jinja2 Templates
- SQLAlchemy
- HTML5 & CSS3 (Bootstrap)
- SQLite (default) or any other SQL database
- Gunicorn (for deployment)
- Dotenv (for environment config)
